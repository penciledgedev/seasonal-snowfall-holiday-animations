Adds festive snowfall and holiday animations to your WordPress site with configurable settings.

== Description ==
This plugin adds beautiful snowfall and twinkling light animations to your WordPress site, giving it a festive holiday look. Customize the number of snowflakes, the speed of the animation, and even add a festive cursor!

== Installation ==
1. Upload the plugin files to the `/wp-content/plugins/seasonal-snowfall-holiday-animations` directory, or install the plugin through the WordPress plugins screen directly.
2. Activate the plugin through the 'Plugins' menu in WordPress.
3. Use the settings under Admin Panel > Settings > Snowfall & Holiday to configure the plugin.

== Frequently Asked Questions ==
= Can I customize the animations? =
Yes, the plugin allows you to choose the type of animation, the count of elements, and their speed.
